# 🚖 OLA Rides Analytics Dashboard

## Project Overview

This Power BI dashboard analyzes OLA ride booking data and provides insights into ride performance, revenue, cancellations, booking status, and vehicle-wise trends.

## Dashboard Preview

![Dashboard](Screenshots/Ola-dashboard.png)

---

## Features

- Total Rides
- Total Revenue
- Completed Rides
- Cancelled Rides
- Booking Status Analysis
- Revenue by Vehicle Type
- Rides by Vehicle Type
- Customer Cancellation Analysis
- Daily Ride Trend
- Interactive Filters
  - Date
  - Vehicle Type
  - Booking Status
  - Payment Method
  - Pickup Location

---

## Tools Used

- Microsoft Power BI
- Excel / CSV
- Power Query
- DAX

---

## KPIs

- Total Rides
- Total Revenue
- Completed Rides
- Cancelled Rides
- Booking Success Rate
- Revenue by Vehicle Type

---

## Insights

- Prime Sedan generated the highest revenue.
- Most rides were successfully completed.
- Customer cancellation rate is relatively low.
- Daily rides remained stable throughout the month.
- eBike and Bike bookings contributed significantly to total rides.
